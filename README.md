## Functionalities
- [ ]  < It uses NLP techniques to pre-process the news-title >
- [ ]  < The input is then taken by LSTM model. >
- [ ]  < The model detects if the news is fake or real >
- [ ]  < The final answer is displayed on web-app using streamlit. >
